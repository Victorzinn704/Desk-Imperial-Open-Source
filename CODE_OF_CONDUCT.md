# Código de Conduta

## Nosso compromisso

O Desk Imperial é um projeto open source voltado para comerciantes brasileiros que precisam de ferramentas de gestão acessíveis. Para que o projeto seja acolhedor para qualquer pessoa que queira contribuir — independente de experiência, origem, identidade ou perspectiva — todos os participantes se comprometem a manter um ambiente respeitoso e colaborativo.

## O que esperamos

- Linguagem respeitosa e construtiva nas discussões
- Críticas focadas no código e nas ideias, não nas pessoas
- Abertura para diferentes pontos de vista e experiências
- Reconhecimento quando errar e disposição para corrigir
- Foco no que é melhor para o projeto e para seus usuários

## O que não é tolerado

- Linguagem ofensiva, discriminatória ou ataques pessoais
- Assédio de qualquer forma, público ou privado
- Publicar informações privadas de outras pessoas sem consentimento
- Comportamento que razoavelmente seria considerado inapropriado em ambiente profissional

## Onde se aplica

Este código se aplica a todos os espaços do projeto: issues, pull requests, discussões, comentários, e qualquer comunicação relacionada ao Desk Imperial.

## Relatar um problema

Se você presenciar ou sofrer comportamento que viola este código, entre em contato com o mantenedor diretamente: **victorzinn704@gmail.com**

Todos os relatos serão tratados com confidencialidade.

## Aplicação

O mantenedor pode remover comentários, fechar issues ou bloquear contribuidores que violem este código, a seu critério, com ou sem aviso prévio.
